变量声明

又叫做css自定义属性哟

正常来说，css变量里面可以放各种值，注意css变量大小写敏感

var()变量

用于读取变量

可以用第二个参数表示默认值

可以变量套变量

变量值的类型

  • 变量值可以是字符串与其他字符串拼接
  • 如果是数值，不能与数值单位直接连用
  • 如果变量值有单位，就不能写成字符串了

作用域

同一个css变量，可以在多个选择器声明，读取的时候，优先级最高的声明生效，与css的层叠规则一样

变量的作用域是他选择器生效的有效范围

for的范围是body选择器的范围，bar的范围是.content的范围

因此全局变量一般放在根元素里面，方便访问修改
